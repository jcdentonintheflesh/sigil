# Welcome to Sigil

A native markdown reader built for your eyes.

## Features

- **Six built-in themes** — Neon, Obsidian, Nord, Parchment, Paper, Terminal
- *Inline editing* with syntax highlighting
- `Cmd+` / `Cmd-` to adjust font size
- Zero network access, zero telemetry

### Code Example

```python
def fibonacci(n):
    a, b = 0, 1
    for _ in range(n):
        a, b = b, a + b
    return a
```
